# Habit Tracker Frontend

A modern full-stack habit tracking frontend built using React.js, Context API, Tailwind CSS, Axios, and React Router.

## Live Demo
[https://habit-tracker-frontend-inky.vercel.app/]

---

# Features

- Add/Delete habits
- Mark habits as completed
- Analytics dashboard
- Weekly productivity chart
- Settings page
- Global state management using Context API
- Responsive UI with Tailwind CSS
- Laravel API integration

---

# Tech Stack

## Frontend
